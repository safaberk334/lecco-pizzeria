# Pizzeria del Lago

A single-page website for an authentic Italian pizzeria located in Lecco, on the shores of Lake Como. Built with modern web technologies and fully responsive design.

![Screenshot](screenshot.png)

## Live Demo

[https://safaberk334.github.io/lecco-pizzeria/](https://safaberk334.github.io/lecco-pizzeria/)

## Features

- Fully responsive layout optimized for all screen sizes
- Interactive menu showcase with categories
- Photo gallery highlighting dishes and ambiance
- Online reservation form
- Embedded Google Maps for easy navigation
- Smooth loading overlay and page transitions

## Getting Started

Clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/safaberk334/lecco-pizzeria.git
cd lecco-pizzeria
```

No build step or package manager required.
